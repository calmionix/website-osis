# OSIS MTs Darul Hikam Cirebon - Website

Website resmi Organisasi Siswa Intra Sekolah (OSIS) MTs Darul Hikam Cirebon.

## 🌟 Fitur Utama

### 📱 Halaman Publik
- **Beranda** - Hero section, statistik, sambutan ketua, preview program
- **Tentang** - Sejarah, visi misi, struktur organisasi
- **Pengurus** - Grid pengurus dengan filter
- **Program** - Daftar program kerja dengan progress
- **Galeri** - Foto dan video kegiatan
- **Event** - Kalender dan daftar event
- **Kontak** - Form kontak aktif
- **Download** - Pusat unduhan dokumen
- **Search** - Pencarian konten
- **404** - Halaman error custom

### 🔐 Fitur Teknis
- **Authentication** - Firebase Google OAuth (khusus email: calmionix@gmail.com)
- **Dark Mode** - Toggle tema gelap/terang dengan localStorage
- **Responsive** - Mobile-first design
- **Glassmorphism Navbar** - Navbar dengan efek blur
- **Smooth Animations** - Scroll animations, hover effects
- **Real-time Clock** - Jam realtime di navbar

### 🎨 Desain
- **Tema**: Islamic Green (#1a5f2a) + Gold (#d4af37)
- **Font**: Poppins (heading) + Inter (body)
- **Style**: Modern, clean, premium UI/UX

## 🛠️ Teknologi

- **HTML5** - Markup semantic
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript** - Vanilla JS
- **Firebase** - Authentication
- **Google Fonts** - Typography
